### Step 1: Check Today's WOD

2. View today's workout:

1. Click **"Check-In"** in navigation
2. You'll see two sections:

**Your Goals**
   - Personal goals set by admin
   - Track your progress
   - Earn bonus points

3. **Answer Each Item**
   - Click **"Yes"** if completed

4. **Submit Check-In**
   - Click **"Submit"** button
   - See total points earned
   - **Note**: Can only check in ONCE per day

## â“ Troubleshooting

### Cannot Login

- **Check credentials**: Email and password correct?
- **Account approved?**: Wait for admin approval
- **Password reset**: Contact admin

### Cannot Check In

- **Already submitted?**: Can only check in once per day
- **All items answered?**: Must answer all questions
- **Technical issue?**: Hard refresh (Ctrl+Shift+R)

### WOD Not Showing

- **Squad assigned?**: Contact admin to assign squad
- **Workout scheduled?**: Admin may not have scheduled WOD yet
- **Refresh**: Hard refresh the page

### Goals Not Showing

- **Admin assignment**: Goals must be assigned by admin
- **Check back**: Admin may assign goals soon
